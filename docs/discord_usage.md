AI Debate System を Discord で利用する方法について説明します。

## 1. 議論を開始する (/discuss)
スラッシュコマンド `/discuss` を使用して、新しいディベートを開始します。

- **コマンド:** `/discuss`
- **引数:**
  - `topic`: 議論したいテーマ（例：「AIは人類を幸せにするか？」）
  - `model` (任意): 最初に発言させたいモデルを選択
- **動作:** コマンドを実行すると、議題専用の新しいテキストチャンネルが自動的に作成され、司会AI（Gemini）が議論を開始します。

## 2. 議論を進行する
議論は以下の順序で自動的に進行します。
1. **司会（Gemini）**が議題を提示。
2. **参加AI（Gemma, Phi, Llama, GPT-OSS-Q2）**が順番に意見を述べる。
3. AIが別のAIをメンション（例：「@Llama さんはどう思いますか？」）した場合、そのAIが次に発言します。
4. 一定ターン経過後、**司会（Gemini）**がまとめを行い、議論が終了します。

## 3. 人間が介入する
議論の途中で人間が意見を挟んだり、特定のAIに指示を出したりすることができます。

### 3.1. スラッシュコマンドでの介入 (/intervene)
- **コマンド:** `/intervene`
- **引数:**
  - `target`: 指示を出したいAIを選択（またはIDを入力）
  - `message`: 指示内容（例：「もっと具体的な事例を挙げて反論してください」）

### 3.2. メンションでの介入
議論チャンネル内でAIをメンションしてメッセージを送信することで、直接そのAIに返答させることができます。

- **例:** `@Gemma さんの意見は少し極端ではないでしょうか？`

## 4. 参加AIモデル一覧
- **Gemini:** 司会・ファシリテーター。議論の開始と終了、まとめを担当。
- **Gemma:** 参加AI。
- **Phi:** 参加AI。
- **Llama:** 参加AI。
- **GPT-OSS-Q2:** 参加AI。

## 5. 注意事項
- チャンネル作成後、AIの最初の応答まで数十秒かかる場合があります。
- 議論が終了したチャンネルでも、人間が介入（メンション等）することで議論を再開させることが可能です。
