# Release Notes

## v1.0.1 (2026-05-04)

### 修正
- `/discuss` コマンド実行時に発生していた `Non-static method App\Jobs\ProcessDebateTurn::dispatch() cannot be called statically` エラーを修正しました。
- 全てのジョブのディスパッチ処理を Laravel のグローバルヘルパー `dispatch()` を使用する形式に統一し、安定性を向上させました。

---

## v1.0

AI Debate System の最初のメジャーリリースです。
複数の AI モデルが Discord 上で議論を行う基盤機能が完成し、人間による介入機能やドキュメントが整備されました。

## 主な新機能と改善
- **人間介入機能の強化**: `/intervene` コマンドによる指示内容が Discord 上で見えるようになり、完了済みのセッションも再開可能になりました。
- **最新環境への対応**: Laravel 13.7.0 および PHP 8.5 への対応を行いました。
- **リファクタリング**: `app` および `tests` ディレクトリ配下を整理し、型安全性の向上とコードの可読性を改善しました。
- **ログと診断の強化**: ジョブの実行状況や API リクエストのデバッグを容易にするため、ログ出力を強化し、疎通確認用コマンドを追加しました。

## ドキュメント
- **設計・運用ドキュメントの整備**: 基本設計、詳細設計、運用手順書、および Discord での使い方のガイドを追加・更新しました。
- **README の刷新**: プロジェクトの目的や貢献方法を明確化しました。
